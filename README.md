# Cara Menjalankan Program

## Soal 1

1. Buka Text Editor
2. Masuk ke folder soal1
3. Buka index.html
4. Klik kanan pilih 'Open with Live Server' (Harus terinstall Ekstensi Live Server )

## Soal 2

1. Buka Text Editor
2. Masuk ke folder soal2
3. Buka terminal
4. jalankan 'npm install'
5. Dapat langusng dijalankan dengan perintah 'npm run dev'
6. Atau dibuild terlebih dahulu dengan 'npm run build'
7. Kemudian jalankan 'npm run start'

## Soal 3

1. Buka Text Editor
2. Masuk ke folder soal3
3. Buka terminal
4. Masukkan perintah 'node app.js'

## Soal 4

1. Buka Text Editor
2. Masuk ke folder soal4
3. Buka terminal
4. Masukkan perintah 'node app.js'
